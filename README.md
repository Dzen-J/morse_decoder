# morse_decoder
Будущее. Вы — марсианин, рожденный и выросший на Красной планете. Уже более полувека ваша колония живёт в полной изоляции: материнская планета Земля внезапно замолчала, перестав отвечать на любые попытки связи. На Марсе растут поколения, которые никогда не слышали голоса родной планеты.

Сегодня многое может измениться. Колония фиксирует странный сигнал: слабый, едва различимый, словно призрак прошлого. Он кажется невероятно древним и звучит словно архаичный код, которым пользовались земляне много веков назад. Этот сигнал ставит множество вопросов:

Что отправило его? Какой секрет он скрывает?
Вернулся ли на связь человек, или это отголоски давно исчезнувшей цивилизации?
Ваша миссия состоит в том, чтобы разгадать тайну этого послания.

# Основные задачи:
Создайте модель машинного обучения, способную распознавать код Морзе из аудиосигналов. В вашем распоряжении архив с данными для обучения — morse_dataset.zip.
Задача со звёздочкой ⭐: Расшифруйте послание, закодированное в последних 17 аудио файлах архива morse_dataset.zip, и раскройте тайну, которую несёт этот сигнал. Что пытаются донести отправители?
Дополнительные детали, которые могут помочь вам в разгадке тайны, доступны в файле logbook.txt.

# Критерии оценки:
Метрика качества:  Levenshtein Mean — усредненное расстояние Левенштейна по тестовому датасету.

Для проверки решения задачи со звёздочкой в файле submission.csv замените ответы модели распознавания на правильные – декодированные вами для последних 17 аудио файлов. Изменение метрики качества подскажет правильность решения. Однако, не ожидайте существенного улучшения метрики, примеров всего 17 из 5000.

# Описание данных:

sample_submission.csv – пример отправки в систему оценки

train.csv – обучающие примеры и расшифровки к ним

test.csv – тестовый список файлов для оценки, совпадает со списком в sample_submission.csv

morse_dataset.zip – zip-файл, содержащий аудиофайлы в формате opus, полученные из исходников с частотой дискретизации 8кГц

logbook.txt – журнал обмена сообщениями колонии

Ссылка на датасет соревнования: https://www.kaggle.com/competitions/morse-decoder/data
